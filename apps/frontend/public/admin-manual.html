Chapter 1

시스템 개요

PMCS는 현장에 설치된 전력품질보상장치(HMI)의 실시간 데이터를 수집·표시하는 웹 기반 모니터링 시스템입니다.

구성 요소

시스템 구성

HMI (현장 장치) — 현장에 설치된 PAF / PSVG / PSTA 장비. 내장 LTE 모뎀으로 주기적으로 데이터를 전송합니다.

백엔드 서버 — HMI 데이터를 수신·저장하고, 프론트엔드에 API를 제공합니다.

웹 대시보드 — 관리자·건설사·현장 담당자가 실시간 현황을 모니터링합니다.

핵심 원칙

HMI는 유심(USIM)의 ICCID 번호만 서버로 전송합니다. 서버는 ICCID를 조회해 어떤 설치지점의 장비인지 식별합니다. 따라서 HMI가 데이터를 보내기 전에 ICCID 매핑이 완료되어 있어야 합니다.

Chapter 3

현장 등록

아파트 단지, 건물 등 납품 현장을 시스템에 등록합니다. 메뉴: 헤더 상단 "현장 관리" 링크 (/admin/sites)

1
현장 관리 페이지 접속

상단 헤더의 "현장 관리" 링크를 클릭합니다. 관리자 계정으로 로그인된 상태여야 합니다.

2
[+ 새 현장 추가] 버튼 클릭

오른쪽 상단의 버튼을 누르면 등록 폼이 펼쳐집니다.

3
현장 정보 입력

아래 필드를 채운 후 "현장 저장" 버튼을 누릅니다.

필드 설명 예시
현장 ID 필수 영문 소문자·숫자·하이픈만 허용. 한 번 생성하면 변경 불가. lotte-dongtan-xi
현장명 필수 대시보드에 표시되는 이름 동탄 롯데캐슬
건설사 필수 드롭다운 선택. 없으면 "직접 입력…" 선택 후 수기 입력 롯데건설
지역 필수 드롭다운 선택 경기도
주소 필수 도로명 주소 (지도 마커 위치 결정에 사용) 경기도 화성시 동탄대로 520
현장 ID 규칙

소문자 영문·숫자·하이픈(-)만 사용 가능합니다. 공백, 한글, 대문자는 사용 불가. 예: hyundai-gumi-hillstate

현장 삭제 주의

현장을 삭제하면 해당 현장의 모든 설치지점·장치·이력 데이터가 영구 삭제됩니다. 삭제 전 반드시 확인하세요.

Chapter 4

설치지점 추가

현장 내 HMI가 설치된 각 위치(변전실, 전기실 등)를 설치지점으로 등록합니다.

1
현장 카드 펼치기

현장 관리 페이지에서 해당 현장 행을 클릭하면 설치지점 목록이 펼쳐집니다.

2
[+ 설치지점 추가] 클릭

현장 카드 하단의 버튼을 클릭합니다.

3
정보 입력 후 [추가] 클릭

설치지점 이름과 (선택적으로) ID를 입력합니다.

필드 설명 예시
설치지점 이름 필수 대시보드에 표시되는 위치명 102동 변전실
ID 선택 비워두면 자동 생성. 직접 지정 시 영문·숫자 권장. PSVG-DONGTAN01
ID 네이밍 권장

ID는 나중에 HMI 펌웨어나 유지보수 이력에서 참조할 수 있으므로 규칙을 통일하는 것을 권장합니다.
예시 패턴: [모델]-[현장약어][번호] → PSVG-DONGTAN01

Chapter 5

ICCID 매핑

HMI에 삽입된 유심(USIM)의 ICCID를 설치지점에 연결합니다. 이 단계가 완료되어야 HMI 데이터가 올바른 설치지점으로 수신됩니다.

ICCID 확인 방법

ICCID는 유심 케이스에 인쇄된 20자리 숫자입니다. HMI 로그에서도 확인 가능합니다:

[ICCID] Got: 89464283446100128110
HMI 시리얼 로그에서 위와 같이 출력됩니다.

매핑 절차

1
설치지점 행에서 ICCID 입력

현장 관리 페이지에서 해당 설치지점 행의 ICCID (USIM) 입력란에 ICCID 번호를 입력합니다.

2
[저장] 버튼 클릭

해당 행의 저장 버튼을 누릅니다. "ICCID가 저장되었습니다." 메시지가 표시되면 성공입니다.

3
HMI 전원 ON / 데이터 수신 확인

HMI를 켜면 약 1~2분 내로 대시보드에 데이터가 표시됩니다.

입력 편의

ICCID에 공백이나 하이픈이 포함되어 있어도 저장 시 자동으로 제거됩니다.
예: 8946 4283 4461 0012 8110 → 89464283446100128110

ICCID 중복 불가

하나의 ICCID는 하나의 설치지점에만 등록할 수 있습니다. 다른 설치지점에 이미 등록된 ICCID를 저장하면 오류가 발생합니다. 기존 매핑을 먼저 제거(빈칸으로 저장)한 후 재등록하세요.

ICCID 제거

ICCID 입력란을 비워두고 저장하면 해당 설치지점의 ICCID 매핑이 해제됩니다. 해제 후 HMI 데이터는 수신되지 않습니다.

Chapter 6

대시보드 보기

메인 페이지(/)에서 전체 현장의 실시간 현황을 확인합니다.

화면 구성

레이아웃

상단 KPI 바 — 전체 장비 수, 정상/대기/이상/오프라인 카운트를 표시합니다.

좌측 사이드바 — 지역 → 현장 → 설치지점 순으로 아코디언으로 표시됩니다.

중앙 지도 — 한국 지도 위에 현장 위치와 상태가 마커로 표시됩니다.

우측 패널 — 선택한 현장의 설치지점별 최신 전력 수치를 표시합니다.

장치 상태 색상

RUNNING  정상 운전 중    STANDBY  대기 중    FAULT  이상 발생    OFFLINE  통신 없음

자동 새로고침

대시보드는 30초마다 자동으로 데이터를 새로고침합니다. 상단 LIVE 인디케이터에서 남은 시간을 확인할 수 있습니다.

지도 사용법

  • 마우스 스크롤로 확대/축소
  • 드래그로 지도 이동
  • 현장 마커 또는 지역 클릭 시 해당 현장이 선택됨
  • 현장 등록 직후 데이터 없으면 OFFLINE 마커로 표시됨
Chapter 7

장치 상세 확인

개별 설치지점의 상세 전력 데이터, 모듈 상태, 24시간 이력 차트를 확인합니다.

1
설치지점 카드 클릭

대시보드 우측 패널 또는 좌측 사이드바에서 설치지점을 클릭합니다.

2
상세 페이지 확인

전압·전류·역률·무효전력·모듈 상태·온도·팬속도 등 실시간 수치와 24시간 이력 차트를 확인합니다.

모듈 상태 코드

코드 의미
0 오프라인 / 미연결
1 대기 (Standby)
2 정상 운전 (Running)
3 이상 / 고장 (Fault)

Chapter 9

문제 해결

데이터가 수신되지 않는 경우

①
ICCID 매핑 확인

현장 관리 페이지에서 해당 설치지점의 ICCID가 저장되어 있는지 확인합니다.

②
HMI 로그 확인

HMI 시리얼 로그에서 [ICCID] Got: ... 라인을 확인합니다. ICCID가 올바르게 읽히는지 확인하세요.

③
서버 응답 확인

HMI 로그에서 +QHTTPPOST: 0,200,... 응답이 있는지 확인합니다. 200이 아니면 네트워크 문제입니다.

④
identity 확인

서버 응답 JSON에서 "identity":{"installationId":null}이면 ICCID 매핑이 저장되지 않은 것입니다. 관리 페이지에서 다시 저장하세요.

대시보드에 현장이 표시되지 않는 경우

확인 사항

현장 관리 페이지에서 현장과 설치지점이 올바르게 등록되어 있는지 확인하세요. 현장을 등록하면 OFFLINE 상태로 즉시 대시보드에 나타납니다. 지도에서 해당 지역을 확인하세요 (주소 기반으로 위치가 결정됩니다).

"unknown" 현장으로 저장되는 경우

원인 및 해결

ICCID 매핑이 저장되지 않은 상태에서 HMI가 데이터를 전송하면 "unknown" 현장으로 임시 저장될 수 있습니다.

해결: 현장 관리에서 설치지점을 등록하고 ICCID를 정확히 저장한 후, HMI를 재시작(또는 다음 전송 주기 대기)하세요.

Chapter 10

신규 현장 설치 체크리스트

납품 전·후로 아래 항목을 순서대로 확인하세요.

사전 준비 (납품 전)

  • HMI 유심 ICCID 번호 확보 (유심 케이스 또는 HMI 로그에서 확인)
  • 현장명, 주소, 건설사 정보 확인
  • 설치지점(변전실, 전기실 등) 목록과 위치 확인

어드민 등록 (납품 당일)

  • 현장 관리 페이지 접속 (/admin/sites)
  • 새 현장 추가 → ID·이름·건설사·지역·주소 입력 → 저장
  • 현장 펼치기 → 설치지점 추가 (변전실·전기실 각각)
  • 각 설치지점에 ICCID 입력 → [저장] 클릭 → 성공 메시지 확인

현장 설치 후 확인

  • HMI 전원 ON 후 약 2분 대기
  • 대시보드에서 해당 현장이 OFFLINE → RUNNING으로 전환 확인
  • 설치지점 상세 페이지에서 전압(V), 전류(A) 수치 정상 확인
  • 모듈 상태 패널에서 각 모듈이 2(Running) 상태인지 확인
  • 24시간 이력 차트에 데이터 적재 시작 확인
완료 기준

대시보드에서 현장 상태가 RUNNING으로 표시되고, 전압·전류 수치가 정상 범위이면 설치가 완료된 것입니다.
